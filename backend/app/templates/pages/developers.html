API Reference

Base URL: /api/v1

Authentication

API endpoints use an API key passed via the X-API-Key header. Key management endpoints use JWT Bearer authentication. 

X-API-Key: nrm_your_api_key_here
POST /api/v1/translate

Translate text between neurotypical and autistic communication styles.

Request Body

Field Type Required
text string Yes
direction string Yes
template string No

direction: autistic_to_neurotypical or neurotypical_to_autistic

template: casual, professional, emotional, technical, conflict

Response (200)

{
  "original_text": "Your message",
  "translated_text": "Translated version",
  "direction": "neurotypical_to_autistic"
}
GET /api/v1/usage

Returns usage statistics for all API keys owned by the authenticated user.

Response (200)

[{
  "api_key_id": "uuid",
  "api_key_name": "My App Key",
  "request_count": 42,
  "rate_limit": 100,
  "last_used_at": "2026-03-21T12:00:00Z"
}]
POST /api/v1/keys

Create a new API key. Requires JWT Bearer authentication.

Request Body

{
  "name": "My Application",
  "rate_limit": 100
}

Response (200)

{
  "id": "uuid",
  "name": "My Application",
  "key": "nrm_full_api_key_shown_once",
  "rate_limit": 100,
  "created_at": "2026-03-21T12:00:00Z",
  "is_active": true,
  "request_count": 0,
  "last_used_at": null
}
GET /api/v1/keys

List all API keys for the authenticated user (key values are masked). Requires JWT Bearer authentication.

Response (200)

[{
  "id": "uuid",
  "name": "My Application",
  "key_prefix": "nrm_abc12345...",
  "rate_limit": 100,
  "created_at": "2026-03-21T12:00:00Z",
  "is_active": true,
  "request_count": 42,
  "last_used_at": "2026-03-21T15:00:00Z"
}]
DELETE /api/v1/keys/{key_id}

Revoke (deactivate) an API key. Requires JWT Bearer authentication. Returns 204 No Content on success.

Rate Limiting

  • Each API key has a configurable rate limit (default: 100 requests/hour)
  • Rate limiting uses a 1-hour sliding window
  • When exceeded, the API returns 429 Too Many Requests

Error Codes

Code Description
400 Invalid request (bad direction, missing fields)
401 Invalid or inactive API key / Missing authentication
404 Resource not found
429 Rate limit exceeded

import requests

BASE_URL = "{{ base_url }}/api/v1"

response = requests.post(
    f"{BASE_URL}/translate",
    headers={
        "Content-Type": "application/json",
        "X-API-Key": API_KEY,

        "direction": "neurotypical_to_autistic",

data = response.json()
print(data["translated_text"])

Manage API Keys

import requests

JWT_TOKEN = "your_jwt_token_here"
BASE_URL = "{{ base_url }}/api/v1"
HEADERS = {"Authorization": f"Bearer {JWT_TOKEN}"}

# Create a new key
resp = requests.post(
    f"{BASE_URL}/keys",
    headers=HEADERS,
    json={"name": "My App", "rate_limit": 100},
)
new_key = resp.json()
print(f"API Key: {new_key['key']}")

# List all keys
resp = requests.get(f"{BASE_URL}/keys", headers=HEADERS)
for key in resp.json():
    print(f"{key['name']}: {key['key_prefix']}")

# Revoke a key
key_id = new_key["id"]
requests.delete(f"{BASE_URL}/keys/{key_id}", headers=HEADERS)
